# Code of Conduct

We want this project to be a respectful, constructive, and welcoming space for everyone.

## Expected Behavior

- Be respectful and considerate in discussions
- Give constructive feedback
- Be welcoming to new contributors
- Focus on collaboration and improving the project

## Unacceptable Behavior

- Harassment, discrimination, or hateful conduct
- Personal attacks, insults, or abusive language
- Trolling, deliberate disruption, or bad-faith behavior
- Sharing someone else’s private information without permission

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and other community interactions related to this project.

## Enforcement

Maintainers may remove, edit, or restrict content and participation that violates this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, please contact the maintainers.
